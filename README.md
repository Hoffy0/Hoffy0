## Projects

**[doto-relay](https://github.com/Hoffy0/doto-relay) — Multi-agent coordination for Claude Code**

Orchestrates multiple Claude Code agents (Orchestrator, Workers, Reviewer) through a shared database — SQLite locally or Supabase/Postgres in the cloud. Each agent picks up exactly where the last one left off: no context loss, no manual handoffs, atomic task claiming. v1.5 ships autonomous Workers via the Claude Agent SDK on June 15.

**Bastet — Home Lab Infrastructure**

Personal bare-metal lab running Proxmox/KVM for high-availability simulation, pfSense for advanced network segmentation (VLANs + firewall rules), and self-hosted containerized services — used as a sandbox for automation and monitoring tooling.
